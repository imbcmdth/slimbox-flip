How It Works

If you drop this script into a page that already uses Slimbox2, it will continue to function exactly as before. The new "flip" feature is only triggered on anchor tags with a rev="...url.." attribute. That means the HTML is still 100% valid markup since we utilize an existing and valid attribute for our purposes.

Example Anchor

<a <!-- Start of (A)nchor Tag -->
href="img/dirty_projectors.jpg" <!-- Required. HREF is the main image to be displayed in the lightbox -->
rev="img/dirty_projectors_back.jpg" <!-- Optional. REV is the flip-side (or REVerse) image to be displayed in the lightbox on "Flip" -->
title="Dirty Projectors, Bitte Orca" <!-- Optional. TITLE to display for this image. -->
rel="lightbox[albumsA]"> <!-- Required (for Slimbox2 functionality.) REL is set to "lightbox" to bind Slimbox2 to this anchor. The text inside the []s tells Slimbox which gallery (albumsA in this case) this image belongs in -->
<img src="img/dirty_projectors_thumb.jpg"/> <!-- Optional. This is the thumbnail to display -->
</a> <!-- End of (A)nchor Tag -->
